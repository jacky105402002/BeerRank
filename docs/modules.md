# Modules

| Module | Responsibility | Notes |
|---|---|---|
| Auth | Third-party login and session state | MVP: Google login preferred |
| Profiles | User-facing identity | Display name, avatar, bio later |
| Beers | Beer catalog records | Can be user-created in MVP |
| Reviews / Posts | Social beer rating posts | Up to 3 photos; public/private visibility |
| Comments | Comment thread | One-level replies for MVP |
| Reactions | Like/unlike posts and optionally comments | Drives social feedback |
| Rankings | Leaderboards | Based on rating and engagement |
| Media | Image upload and serving | Supabase Storage later; mock URLs first |

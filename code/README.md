# BeerRank Code Workspace

Application code lives here.

## Apps

- `apps/web`: React + TypeScript + Vite frontend.
- `apps/api`: NestJS TypeScript API scaffold.

## Packages

- `packages/shared`: shared DTOs, constants, and types.

## Local Ports

- Web: `http://localhost:6677`
- API: `http://localhost:3001/api`
